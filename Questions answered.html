What is internet?

The internet is a connection of computers across the globe through a network from which we can share information

What is browser?

URL:- Uniform resource Locator = Path of a website
Domain:- Name of a site
Subdomains:- WWW
Main Domains:- main searched website or the name of the website
Top level Domains:- .in,.org, Etc...,
IP address:- Identity of a device that is using internet.

Examples:Chrome, Google, Opera

HTML(Not case sensitive):

Hyper Text Markup Language is a language used to communicate with the browser using a visual interface. It is not case sensitive because it displays the output despite errors.

XHTML(Case sensitive):

It is case sensitive because any error in the code causes it to not display the output.
